# 황도 12궁 별자리 딥리서치 요청

## 목적
엔터테인먼트 앱 운세 콘텐츠용 별자리 데이터 수집

## 중요 조건
- **엔터테인먼트/재미 목적** (점술 서비스 X)
- **긍정적/희망적 해석 중심**
- **20대 타겟** 친근한 MZ 말투
- **밈/트렌드** 적절히 반영

---

## 리서치 요청 1: 12별자리 기본 정보

각 별자리별로 다음 정보를 정리해주세요:

### 별자리 정보 템플릿
```
별자리명 (영문)
- 기간: MM/DD ~ MM/DD
- 원소: 불/흙/공기/물
- 특성: 활동궁/고정궁/변통궁
- 대표 이모지: 1개
- 수호성: (한글명)
- 키워드: 5개
- 성격 요약: 2~3문장 (긍정적, 재미있게)
- 장점: 3가지
- 성장 포인트: 2가지 (단점 대신 "성장하면 더 좋은" 관점)
- 연애 스타일: 2~3문장
- 궁합 좋은 별자리: 3개 (이유 포함)
- 2025년 키워드: 3개 (을사년 관련 있으면 연결)
```

### 12별자리 목록
1. Aries 양자리 (3/21~4/19)
2. Taurus 황소자리 (4/20~5/20)
3. Gemini 쌍둥이자리 (5/21~6/21)
4. Cancer 게자리 (6/22~7/22)
5. Leo 사자자리 (7/23~8/22)
6. Virgo 처녀자리 (8/23~9/22)
7. Libra 천칭자리 (9/23~10/22)

11. Aquarius 물병자리 (1/20~2/18)
12. Pisces 물고기자리 (2/19~3/20)

## 리서치 요청 2: 오늘의 운세 메시지 템플릿

각 별자리별 운세 메시지 10개씩 (다양한 상황):

### 카테고리별 2개씩
1. **전체운** (일반적인 하루)
2. **연애운** (솔로/커플 공통)
3. **재물운** (소소한 재물, 투자조언 X)
4. **건강운** (컨디션, 에너지 관점)
5. **행운의 팁** (오늘의 럭키 아이템/색상/숫자)

### 예시 (양자리)
```
전체운:
- "오늘의 양자리는 불타오르는 에너지! 🔥 하고 싶은 거 다 해도 되는 날"
- "잠깐 멈춰서 숨 고르기가 필요한 하루예요. 급한 결정은 내일로!"

연애운:
- "솔로라면? 오늘 만나는 사람 주목! 예상치 못한 설렘이 올지도 💕"
- "연인과 사소한 다툼 주의. 한 발 물러서면 오히려 가까워져요"

재물운:
- "충동구매 위험! 장바구니에 담아만 두세요 🛒"
- "예상치 못한 작은 보너스가 올 수 있어요. 커피값 정도? ☕"

건강운:
- "에너지 과부하 조심! 적당한 휴식이 내일의 폭발력을 만들어요"
- "머리보다 몸이 먼저 반응하는 날. 가벼운 운동 추천!"

행운의 팁:
- "럭키컬러: 빨강 🔴 / 럭키넘버: 7 / 럭키아이템: 빨간 양말"
- "점심은 매운 음식으로! 🌶️ 오후 활력이 달라질 거예요"
```

## 리서치 요청 3: 별자리 밈/특징 정리

MZ세대가 공감하는 별자리별 특징 5개씩:

### 예시 (양자리)
```
1. "일단 저지르고 본다" (충동력 MAX)
2. 화났다가 5분 뒤 언제 그랬냐는 듯 (분노 리셋 빠름)
3. "내가 해볼게" 먼저 말하는 타입
4. 기다리는 거 진짜 못 참음 (배달 5분 지연에 전화함)
5. 좋아하면 티 숨기기 불가능 (얼굴에 다 씀)
```

## 주의사항

### 사용 금지 표현
- "반드시", "틀림없이" 등 단정적 표현
- 건강/질병 예측
- 재물 손실 예측
- 이별/파국 단정
- 특정 직업/적성 단정

### 권장 표현
- "~일 수도 있어요"
- "~의 기운이 느껴져요"
- "~를 시도해보는 건 어떨까요?"
- "~의 가능성이 열려 있어요"
- "~라고 하네요" (출처 언급 느낌)

## 출력 형식

JSON 또는 마크다운 테이블 형식으로 정리해주세요.

```typescript
interface Constellation {
  id: string;              // 'aries'
  nameKo: string;          // '양자리'
  nameEn: string;          // 'Aries'
  emoji: string;           // '♈'
  period: {
    start: string;         // '03-21'
    end: string;           // '04-19'
  };
  element: 'fire' | 'earth' | 'air' | 'water';
  modality: 'cardinal' | 'fixed' | 'mutable';
  rulingPlanet: string;    // '화성'
  keywords: string[];
  personality: {
    summary: string;
    strengths: string[];
    growth: string[];
  };
  love: {
    style: string;
    compatible: string[];  // ['leo', 'sagittarius', 'gemini']
  };
  year2025: {
    keywords: string[];
    message: string;
  };
  dailyMessages: {
    general: string[];
    love: string[];
    money: string[];
    health: string[];
    tips: string[];
  };
  memes: string[];         // MZ 공감 특징
}
```
