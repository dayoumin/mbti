# 타로 메이저 아르카나 22장 딥리서치 프롬프트

## ChatGPT / Gemini / Perplexity 용

---

### 프롬프트

```
MZ세대(20-30대) 타겟 모바일 엔터테인먼트 앱용 타로 카드 데이터를 만들어주세요.

## 요청 사항

메이저 아르카나 22장(0번 The Fool ~ 21번 The World) 각각에 대해:

### 1. 기본 정보
- 카드 번호 (0-21)
- 영문명 / 한글명
- 대표 이모지 1개
- 핵심 키워드 3-5개 (해시태그 형식)

### 2. 정방향 해석 (MZ 말투로)
- 일반 운세 (1-2문장)
- 연애운 (1문장)
- 재물운 (1문장)
- 오늘의 조언 (1문장)

### 3. 역방향 해석
- 일반 해석 (1-2문장)
- 조언 (1문장)

### 4. MZ 감성 요소
- 한줄 요약 (15자 이내, 인스타 공유용)
- 공감 상황 ("이 카드 뽑은 날 나:" 형식)
- 추천 행동 (오늘 해볼 간단한 액션)

## 중요 가이드라인

### 필수 리프레이밍 (부정적 카드 → 긍정적 해석)
- 13번 Death: "죽음" 대신 "변화와 재생, 새로운 챕터의 시작"
- 15번 The Devil: "악마" 대신 "내면의 욕망 인식, 자기 이해의 기회"
- 16번 The Tower: "파괴" 대신 "뜻밖의 돌파구, 답답함 해소"
- 12번 The Hanged Man: "고난" 대신 "새로운 관점, 시야의 전환"
- 18번 The Moon: "불안" 대신 "직관의 시간, 내면 탐구"

### 절대 금지 표현
- 죽음/수명 직접 언급 (죽, 사망, 수명, 단명)
- 질병 언급 (암, 우울증, 공황)
- 금융 투자 조언 (주식, 코인, 로또)
- 법률 문제 (소송, 감옥)
- 단정적 표현 (100%, 확실히, 반드시, 틀림없이)

### 톤앤매너
- MZ 친화적 말투 (밈, 유행어 적절히)
- 긍정적이고 가벼운 톤
- 성찰과 자기 이해 강조
- 성별/연령 고정관념 배제
- "~할 수 있어요", "~해보세요" 식의 열린 표현

## 출력 형식

각 카드를 다음 JSON 형식으로:

{
  "number": 0,
  "id": "the-fool",
  "nameKo": "바보",
  "nameEn": "The Fool",
  "emoji": "🎒",
  "keywords": ["#새로운시작", "#순수", "#모험", "#자유"],
  "upright": {
    "general": "계획은 잠시 접어두고 마음이 이끄는 대로 첫발을 떼보세요!",
    "love": "설레는 새로운 만남이나 관계의 시작을 암시해요.",
    "money": "작은 투자나 새로운 시도가 좋은 기회가 될 수 있어요.",
    "advice": "오늘은 생각보다 행동! 일단 시작해보세요."
  },
  "reversed": {
    "general": "무모한 결정은 잠시 멈추고, 한 번 더 생각해볼 타이밍이에요.",
    "advice": "용기와 무모함의 차이를 구분해보세요."
  },
  "mz": {
    "oneLiner": "일단 저질러!",
    "situation": "YOLO 외치며 휴가 예약 버튼 누르는 중",
    "action": "오늘 미뤄둔 일 중 하나 시작하기"
  }
}

22장 전체를 배열로 출력해주세요.
```

### 추가 질문 프롬프트 (필요시)

```
위에서 만든 타로 카드 데이터를 바탕으로:

1. 카드별 "투표 질문" 5개 만들어주세요.
   예: "가장 YOLO 기질인 카드는?" → 투표 선택지 4개

2. 카드 조합별 스토리 3개 만들어주세요.
   예: "The Fool + The Lovers = 운명적 만남의 시작"

3. 각 카드의 "오늘의 럭키 아이템" 추가해주세요.
   예: The Fool → "새 노트, 여행용 가방"
```
