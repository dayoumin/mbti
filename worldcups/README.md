## 🎯 월드컵 폴더 구조

각 월드컵 폴더는 다음과 같은 구조를 가집니다:

```
worldcup-name/
├── README.md              # 프로젝트 개요 및 설명
├── candidates.md          # 후보 목록 및 선정 근거
├── data/                  # JSON 데이터 파일
│   └── items.json        # 월드컵 아이템 데이터
└── images/               # 썸네일 이미지 자료
    └── ...
```

## 📝 작성 규칙

1. **README.md**: 프로젝트 개요, 선정 기준, 장르 분포 등
2. **candidates.md**: 전체 후보 목록과 출처 자료
3. **data/**: 웹 애플리케이션에서 사용할 JSON 데이터
4. **images/**: 각 항목의 썸네일 이미지 (저작권 주의)
