# 한국 웹툰 월드컵 (32강)

## 📋 프로젝트 개요

2025년 기준 한국의 인기 웹툰 32편을 대결시키는 월드컵 프로젝트입니다.

## 📁 폴더 구조

```
korean-webtoon/
├── README.md              # 프로젝트 설명
├── candidates.md          # 후보 목록 및 출처
├── data/                  # 데이터 파일
│   └── webtoons.json     # 웹툰 데이터 (제목, 작가, 장르, 플랫폼 등)
└── images/               # 웹툰 썸네일 이미지
```

## 🎯 선정 기준

- **네이버웹툰**: 2025년 현재 인기 순위 상위작 (11편)
- **카카오웹툰/다음웹툰**: 역대 명작 및 인기작 (6편)
- **역대 명작 & 인기작**: 문화적 영향력이 큰 작품 (15편)
