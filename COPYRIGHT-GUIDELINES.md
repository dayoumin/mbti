# 저작권 가이드라인

> 본 문서는 MBTI 프로젝트 전체에 적용되는 저작권 가이드라인입니다.

## ⚠️ 저작권 주의사항

### 이미지 및 콘텐츠 사용 제한

**❌ 금지 사항:**
- 웹툰/애니메이션/영화/드라마 등의 이미지를 직접 다운로드하여 서버에 저장
- 공식 포스터/썸네일을 무단으로 복사 및 재배포
- 작품의 스크린샷, 장면 캡처 이미지 사용
- 2차 창작물 무단 사용
- 연예인/유명인 사진 무단 사용
- 로고, 브랜드 이미지 무단 사용

**✅ 허용 가능한 방법:**
- **제목과 텍스트만 표시** (가장 안전)
- **공식 플랫폼 링크**만 제공 (네이버웹툰, 카카오웹툰, MyAnimeList 등)
- 출처 명시와 함께 링크 연결
- 직접 제작한 텍스트 카드 디자인

### 현재 구현 방식

1. **제목 + 장르 표시**
   - 작품 제목 (한글/영문)
   - 장르 정보
   - 간단한 설명

2. **공식 링크 제공**
   - 네이버웹툰: `https://comic.naver.com/webtoon/list?titleId=XXX`
   - 카카오웹툰: `https://webtoon.kakao.com/content/XXX`
   - MyAnimeList: `https://myanimelist.net/anime/XXX`

3. **출처 명시**
   - 모든 데이터에 출처 표기
   - 공식 플랫폼 크레딧 표시

## 📋 배포 전 저작권 체크리스트

> 💡 **전체 배포 체크리스트는 [DEPLOYMENT-CHECKLIST.md](DEPLOYMENT-CHECKLIST.md) 파일을 참고하세요.**

### 개발 단계
- [ ] 이미지 파일이 로컬/서버에 저장되어 있지 않은지 확인
- [ ] 이미지 URL이 외부 직접 링크가 아닌지 확인
- [ ] 모든 작품에 출처가 명시되어 있는지 확인
- [ ] 작품 데이터가 공개된 정보만 포함하는지 확인

### 배포 전 필수 체크
- [ ] **이미지 사용 여부 최종 확인**
  - 사용 안 함: 텍스트만 사용 ✅
  - 사용함: 저작권 검토 필요 ⚠️

- [ ] **링크 방식 확인**
  - 공식 플랫폼 링크만 제공 ✅
  - 직접 임베드는 피하기 ⚠️

- [ ] **출처 표기 확인**
  - 모든 페이지에 출처 명시
  - "데이터 출처: 네이버웹툰, 카카오웹툰 등" 표기

- [ ] **면책 조항 추가**
  - "비상업적 목적의 팬 프로젝트"
  - "모든 저작권은 원저작자에게 있음"
  - "문제 시 즉시 삭제 조치"

### 운영 단계
- [ ] 저작권자로부터 이의 제기 시 즉시 대응 체계 마련
- [ ] 삭제 요청 시 24시간 내 조치 가능하도록 준비
- [ ] 정기적으로 저작권 관련 법규 변경사항 확인

## 🔒 권장 구현 방식

### 1. 텍스트 기반 월드컵 (가장 안전)
```
┌─────────────────────┐
│  나 혼자만 레벨업    │
│  장르: 판타지 액션   │
│  작가: 추공, 현군    │
│  [자세히 보기 →]     │
└─────────────────────┘
```

### 2. 공식 링크 제공
```javascript
// 클릭 시 공식 페이지로 이동
<a href="https://comic.naver.com/webtoon/list?titleId=XXX"
   target="_blank"
   rel="noopener noreferrer">
  나 혼자만 레벨업 - 네이버웹툰에서 보기
</a>
```

### 3. 간단한 색상 카드
```
각 작품마다 고유 색상으로 카드 디자인
텍스트 정보만 표시
이미지 없이 깔끔한 UI
```
