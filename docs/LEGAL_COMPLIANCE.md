## 1. MBTI 상표권/저작권 현황

### 1.1 상표권 소유

| 상표 | 소유자 | 상태 |
|------|--------|------|
| MBTI® | Myers & Briggs Foundation | 등록상표 |
| Myers-Briggs® | Myers & Briggs Foundation | 등록상표 |
| Myers-Briggs Type Indicator® | Myers & Briggs Foundation | 등록상표 |
| 16가지 유형 코드 (ENFP, ISTJ 등) | - | 상표는 아니지만 연관성 문제 |

### 1.2 보호 범위

**저작권으로 보호되는 것:**
- MBTI 검사 문항 원본
- 유형별 공식 설명/해석
- 점수 산출 방식
- 프로파일, 리포트, 차트, 표
- 매뉴얼, 브로셔, 서적의 텍스트

**상표권으로 보호되는 것:**
- "MBTI", "Myers-Briggs" 명칭
- 공식 로고
- 관련 제품/서비스명

### 1.3 위반 시 위험

> — The Myers-Briggs Company

- 법적 소송

## 2. 피해야 할 사항 (체크리스트)

### 2.1 용어 금지 목록

**절대 사용 금지:**
```
❌ MBTI
❌ Myers-Briggs
❌ 16가지 유형 코드 (ENFP, ISTJ, INFJ 등 모든 4글자 조합)
❌ E/I, S/N, T/F, J/P 대조쌍 용어
```

**연관성 있어 피해야 할 용어:**
```
⚠️ 외향(Extraversion) vs 내향(Introversion) - 이 대조쌍 직접 사용
⚠️ 감각(Sensing) vs 직관(Intuition)
⚠️ 사고(Thinking) vs 감정(Feeling)
⚠️ 판단(Judging) vs 인식(Perceiving)
⚠️ "16가지 성격 유형"
⚠️ "성격 유형 검사" (MBTI를 연상시킴)
```

### 2.2 문항 금지 패턴

**MBTI식 대조쌍 질문 금지:**
```
❌ "파티에서 많은 사람과 어울리기 vs 혼자 책 읽기"
❌ "계획적으로 행동 vs 즉흥적으로 행동"
❌ "논리적 분석 vs 감정적 판단"
❌ "세부사항 집중 vs 큰 그림 파악"
```

### 2.3 금지 콘텐츠

```
❌ MBTI 유형별 공식 설명 번역/의역
❌ MBTI 궁합표 복제
❌ MBTI 관련 그래픽/차트 모방
❌ "당신의 MBTI는?" 같은 마케팅 문구
```

## 3. 안전한 대안: 사용 가능한 프레임워크

### 3.1 Big Five (OCEAN) - ✅ 공개 도메인

**특징:**
- 학술적으로 가장 검증된 성격 모델
- **IPIP (International Personality Item Pool)** - 완전 공개 도메인
- 상업적 사용 자유

**5가지 요인:**
| 영문 | 한글 | 설명 |
|------|------|------|
| **O**penness | 개방성 | 새로운 경험에 대한 열림 |
| **C**onscientiousness | 성실성 | 조직화, 목표지향 |
| **E**xtraversion | 외향성 | 사회적 에너지 |
| **A**greeableness | 우호성 | 협조, 신뢰 |
| **N**euroticism | 신경성 | 정서적 안정성 |

**출처:** [bigfive-test.com](https://bigfive-test.com/) (오픈소스)

### 3.2 HEXACO - ✅ 공개 도메인 버전 있음

**특징:**
- Big Five에 정직-겸손(Honesty-Humility) 추가
- **IPIP-HEXACO** 버전은 공개 도메인
- 논문: Ashton, Lee, & Goldberg (2007)

**6가지 요인:**
| 영문 | 한글 |
|------|------|
| **H**onesty-Humility | 정직-겸손 |
| **E**motionality | 정서성 |
| e**X**traversion | 외향성 |
| **A**greeableness | 우호성 |
| **C**onscientiousness | 성실성 |
| **O**penness | 개방성 |

### 3.3 펫 전용 - ✅ 학술 연구 기반 자체 개발

**고양이 (Feline Five):**
- Litchfield et al. (2017) 연구 기반
- 5가지 특성을 자체 용어로 재명명 가능

**강아지 (C-BARQ 기반):**
- 공개된 행동 특성 활용
- 자체 질문 개발

## 4. 새로운 브랜딩 전략

### 4.1 서비스명 후보

**종합 서비스:**
| 이름 | 설명 | 장점 |
|------|------|------|
| **소울메이트 테스트** | 나와 맞는 사람/펫 찾기 | 직관적 |
| **케미 체커** | 관계 케미 분석 | 트렌디 |
| **마음 프로파일** | 성격 프로파일링 | 부드러움 |
| **성격 레이더** | 다각도 성격 분석 | 시각적 |

**고양이 전용:**
| 이름 | 콘셉트 |
|------|--------|
| 냥벤처 지수 | 모험성/호기심 |
| 냥친력 레이더 | 집사 친화도 |
| 냥플레이 프로파일러 | 놀이 취향 |
| 냥성격 올인원 | 종합 |

**강아지 전용:**
| 이름 | 콘셉트 |
|------|--------|
| 멍프로파일 | 종합 성격 |
| 댕댕 케미 | 주인-반려견 궁합 |
| 멍스타일 체커 | 생활/놀이 스타일 |

**사람 전용:**
| 이름 | 콘셉트 |
|------|--------|
| 마음 스펙트럼 | 다차원 성격 |
| 관계 케미 | 인간관계 궁합 |
| 소셜 레이더 | 사회적 스타일 |

### 4.2 성격 차원 재명명

**MBTI 용어 → 안전한 대안:**

| 기존 (금지) | 새로운 이름 | 측정 내용 |
|-------------|-------------|-----------|
| E/I (외향/내향) | **에너지 방향** 또는 **소셜 스타일** | 사회적 에너지 |
| S/N (감각/직관) | **정보 수집** 또는 **인식 스타일** | 정보 처리 방식 |
| T/F (사고/감정) | **결정 방식** 또는 **판단 스타일** | 의사결정 기준 |
| J/P (판단/인식) | **생활 방식** 또는 **계획 스타일** | 생활 구조화 |

**펫 전용 차원:**

| 고양이 (Feline Five 기반) | 강아지 (C-BARQ 기반) |
|---------------------------|----------------------|
| 탐험 지수 (Extraversion) | 활력 지수 (Energy) |
| 경계 지수 (Neuroticism) | 친화 지수 (Sociability) |
| 주도 지수 (Dominance) | 집중 지수 (Training Focus) |
| 변덕 지수 (Impulsiveness) | 끈기 지수 (Perseverance) |
| 친화 지수 (Agreeableness) | 대담 지수 (Boldness) |

### 4.3 결과 라벨 (유형 대신)

**기존 (금지):** ENFP, ISTJ 등 4글자 코드

**새로운 방식 - 설명적 라벨:**

```
사람:
- "열정적 탐험가" (높은 외향성 + 높은 개방성)
- "조용한 분석가" (낮은 외향성 + 높은 성실성)
- "따뜻한 조력자" (높은 우호성 + 높은 정서성)

고양이:
- "호기심 대장 냥이" (높은 탐험 지수)
- "안락한 소파러 냥이" (낮은 탐험 지수)
- "츤데레 보스 냥이" (높은 주도 지수)

강아지:
- "에너지 폭발 댕댕이" (높은 활력 지수)
- "차분한 힐러 댕댕이" (낮은 활력 지수)
- "충성파 수호견" (높은 친화 지수)
```

### 4.4 점수 표현

**기존 (금지):** E 70% / I 30%

**새로운 방식:**
```
소셜 에너지: ████████░░ 80점
탐험 성향:   ██████░░░░ 60점
결정 스타일: ████░░░░░░ 40점
계획 성향:   ██████████ 100점
```

## 5. 문항 설계 원칙 (법적 안전)

### 5.1 DO (권장)

```
✅ 구체적 행동 묘사
   "새로운 장소에 가면 먼저 구석구석 탐색한다"

✅ 관찰 가능한 상황
   "낯선 소리가 나면 숨는다 / 확인하러 간다"

✅ 단일 행동/상황만 질문
   "간식을 주면 바로 먹는다 / 냄새부터 맡는다"

✅ 중립적 표현
   두 선택지 모두 긍정적/중립적으로
```

### 5.2 DON'T (금지)

```
❌ MBTI식 대조쌍
   "계획적 vs 즉흥적" ← 금지
   "논리 vs 감정" ← 금지

❌ 추상적 성향 질문
   "당신은 외향적입니까?" ← 금지

❌ 기존 MBTI 문항 변형/의역
   파티 상황, 모임 상황 등 전형적 MBTI 질문
```

### 5.3 예시 비교

| 금지 (MBTI식) | 안전한 대안 |
|---------------|-------------|
| "파티에서 많은 사람과 어울리기 vs 혼자" | "새로운 모임에서 처음 30분 동안 몇 명과 대화하나요? [0-2명 / 3-5명 / 6명 이상]" |
| "계획적 vs 즉흥적" | "여행 가기 전 숙소를 예약하는 시점은? [2주 전 / 1주 전 / 당일]" |
| "논리 vs 감정" | "친구가 새 옷을 샀는데 별로일 때? [솔직히 말한다 / 다른 장점을 찾는다]" |

## 6. 최종 체크리스트

### 배포 전 필수 확인

**용어 검사:**
- [ ] "MBTI" 단어 없음
- [ ] "Myers-Briggs" 없음
- [ ] 4글자 유형 코드 없음 (ENFP, ISTJ 등)
- [ ] E/I, S/N, T/F, J/P 직접 대조 없음

**문항 검사:**
- [ ] MBTI식 대조쌍 질문 없음
- [ ] 기존 MBTI 문항과 유사한 상황 없음
- [ ] 행동/관찰 중심 문항만 사용

**브랜드 검사:**
- [ ] 서비스명에 MBTI 연상 없음
- [ ] UI/마케팅에 관련 표기 없음
- [ ] 결과 화면에 유형 코드 없음

**기록 보관:**
- [ ] 문항 작성자/날짜 기록
- [ ] 수정 내역 보관
- [ ] "독자 개발" 증빙 문서
