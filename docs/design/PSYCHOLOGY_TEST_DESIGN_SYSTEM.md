# 심리 테스트 디자인 시스템

> 과학적 근거 기반의 심리 테스트 설계 가이드라인 및 확장 전략

---

## 1. 개요

### 1.1 목적

이 문서는 심리 테스트 콘텐츠를 설계할 때 필요한:
- **과학적 근거** (신뢰도, 타당도, 최적 문항 수)
- **바이럴 요소** (공유 유도, 사용자 경험)
- **확장 전략** (새로운 테스트 주제, 다른 분야 적용)

을 종합적으로 다룹니다.

### 1.2 핵심 원칙

| 원칙 | 설명 | 과학적 근거 |
|------|------|-------------|
| **간결성** | 5~7분 내 완료 | 7분 초과 시 이탈률 5~20% 증가 |
| **균형성** | 차원당 7~10개 문항 | 신뢰도 α≥0.80 달성 기준 |
| **공유성** | 결과가 "나를 표현"해야 함 | 바이럴의 핵심 동기 |
| **재미** | 엔터테인먼트 + 인사이트 | 직장인 휴식시간 대화 소재 |

## 2. 과학적 근거 요약

### 2.1 문항 수와 신뢰도

| 차원당 문항 | 내적 일관성 (α) | 권장 | 출처 |
|------------|----------------|------|------|
| 2개 | 0.12~0.86 (평균 0.53) | ❌ 부족 | [TIPI Scoping Review](https://pmc.ncbi.nlm.nih.gov/articles/PMC10330951/) |
| 3~4개 | 0.72+ | ⚠️ 최소 | [BFI-18](https://www.sciencedirect.com/science/article/pii/S0010440X24000658) |
| 6개 | 0.77+ | ✅ 단축형 권장 | [BFI-2-S](https://www.colby.edu/academics/departments-and-programs/psychology/research-opportunities/personality-lab/the-bfi-2/) |
| 7~10개 | 0.80+ | ✅✅ 최적 | [PMC G-Theory 연구](https://pmc.ncbi.nlm.nih.gov/articles/PMC4236721/) |
| 12개 | 0.83+ | 🎯 정밀 측정 | [BFI-2](https://www.colby.edu/academics/departments-and-programs/psychology/research-opportunities/personality-lab/the-bfi-2/) |

### 2.2 완료 시간과 이탈률

| 시간 | 완료율 | 권장 |
|------|--------|------|
| 2~3분 (10문항) | 89% | ✅ 최적 |
| 5~7분 (15~20문항) | 70~80% | ✅ 권장 |
| 10분+ (30문항+) | 50~60% | ⚠️ 심화 전용 |
| 15분+ | 37% 이하 | ❌ 피해야 함 |

**출처**: [PMC Survey Fatigue](https://pmc.ncbi.nlm.nih.gov/articles/PMC11833437/), [Formaloo](https://www.formaloo.com/blog/how-long-should-a-survey-be)

### 2.3 표준 심리 도구 참조

| 도구 | 대상 | 문항 수 | 차원 | 신뢰도 |
|------|------|---------|------|--------|
| Big Five (BFI-2) | 사람 | 60 | 5 | 0.83+ |
| Feline Five | 고양이 | 48 | 5 | 0.72~0.90 |
| C-BARQ | 강아지 | 100 (단축형 42) | 14 | 0.72~0.90 |
| ASQ-SF (애착) | 관계 | 29 | 5 | 양호 |
| TIPI (초단축) | 사람 | 10 | 5 | 낮음 (재검사는 양호) |

## 3. 바이럴 콘텐츠 설계

### 3.1 사람들이 공유하는 4가지 이유

[포레스트 바이럴 사례](https://www.careet.net/Content/View/31) 분석:

| 동기 | 설명 | 테스트 설계 적용 |
|------|------|------------------|
| **자기표현** | 나의 가치관/성격을 보여줌 | 결과에 "나를 대변하는" 문구 |
| **이미지 메이킹** | 원하는 모습 연출 | 긍정적이면서 독특한 결과 |
| **감정 공유** | 놀람, 기쁨, 공감 | 예상 밖의 인사이트 제공 |
| **대화 소재** | 점심시간 수다거리 | 친구/동료와 비교 가능한 결과 |

### 3.2 바이럴 성공 사례

```
포레스트 앱 심리테스트:
- 네이버 실검 1위
- 이틀간 평소의 50배 다운로드
- 이후에도 5~10배 유지
```

**핵심 요소**:
- MZ세대 타겟
- SNS 공유 최적화 (인스타 스토리, 카톡)
- 짧은 테스트 (5분 이내)
- 공유하고 싶은 결과 문구

### 3.3 직장인 점심시간 활용

[트렌드모니터 조사](https://trendmonitor.co.kr/):

| 점심시간 활동 | 비율 |
|--------------|------|
| 가벼운 산책 | 49.3% |
| 낮잠/휴식 | 35.2% |
| **동료와 수다** | 31.1% (증가 추세) |
| 자기계발 | 증가 중 |

**시사점**:
- 점심시간 5분 내 완료 가능한 테스트
- 동료와 결과 비교 가능
- 업무 얘기 NO, 가벼운 재미 YES
- "나도 해볼래!" 유도

**출처**: [테스트모아](https://testmoa.com/), [푸망](https://poomang.com/), [99acorns](https://www.99acorns.com/popular-personality-quizzes-in-2025-whats-trending/)

| 카테고리 | 예시 | 과학적 기반 | 바이럴 잠재력 |
|----------|------|-------------|---------------|
| **성격 유형** | MBTI, 에니어그램 | ✅ 높음 | ⭐⭐⭐⭐⭐ |
| **연애/관계** | 이상형, 애착유형, 연애 스타일 | ✅ 애착이론 | ⭐⭐⭐⭐⭐ |
| **직장/커리어** | 업무 스타일, 리더십, 번아웃 | ✅ 산업심리학 | ⭐⭐⭐⭐ |
| **트렌드/밈** | 신조어, 2024 밈 능력고사 | ❌ 재미 중심 | ⭐⭐⭐⭐⭐ |
| **펫/동물** | 고양이 성격, 강아지 성격 | ✅ Feline Five, C-BARQ | ⭐⭐⭐⭐ |
| **퍼스널컬러** | 봄웜/쿨톤 등 | ⚠️ 일부 근거 | ⭐⭐⭐⭐ |
| **음식/라이프** | 나와 맞는 음식, 여행 스타일 | ❌ 재미 중심 | ⭐⭐⭐ |
| **AI 관련** | 나의 AI 페르소나 | ❌ 트렌드 | ⭐⭐⭐⭐ |

### 5.2 확장 후보 상세

#### A. 이상형 테스트 (연애/관계)

```
과학적 기반: 애착 이론 (Attachment Theory)
- Secure, Anxious, Avoidant, Disorganized
- 출처: ASQ-SF (29문항), ECR-R

테스트 설계:
- "나의 연애 스타일은?" → 애착 유형 분석
- "이상형 찾기" → 선호 성격 + 나와의 궁합
- 결과: "당신과 잘 맞는 상대 유형"

예상 문항:
- "연인이 바쁘다고 연락이 없을 때?"
  A: 이해하고 기다린다
  B: 불안해서 먼저 연락한다
  C: 나도 연락 안 한다
```

#### B. 직장인 업무 스타일 테스트

```
과학적 기반:
- Big Five (특히 Conscientiousness, Neuroticism)
- 직무 적합성 연구

테스트 설계:
- "나의 업무 부캐는?"
- "회사에서 나는 어떤 동물?"

타겟: 점심시간 5분 테스트
결과: 동료와 비교 가능한 유형
```

#### C. 번아웃 체크 테스트

```
과학적 기반:
- MBI (Maslach Burnout Inventory)
- 정서적 소진, 비인격화, 개인적 성취감 저하

테스트 설계:
- 가벼운 버전으로 자가 체크
- 결과에 휴식 팁 제공
- 공유보다는 자기 인사이트 중심
```

#### D. 케미 확장: 다른 동물

```
후보:
- 햄스터/기니피그 (소동물)
- 앵무새/새 (조류)
- 토끼

연구 기반:
- 소동물/조류 성격 연구는 제한적
- C-BARQ/Feline Five 프레임워크 참고하여 설계
```

### 5.3 주제별 우선순위 매트릭스

```
                    바이럴 잠재력
                    높음 ←―――――→ 낮음
                      │
    과학적    높음    │  ① 이상형/애착    ③ 번아웃 체크
    기반             │  ② 업무 스타일
                     │
              낮음   │  ④ 트렌드 테스트   ⑤ 음식/여행
                     │  ⑥ AI 페르소나
                     ↓

우선순위:
1. 이상형/연애 스타일 (애착 이론 기반)
2. 직장인 업무 스타일 (점심시간 타겟)
3. 번아웃 셀프체크 (실용성 높음)
```

## 6. 다른 AI 검토용 요약

### 6.1 프로젝트 개요

```
프로젝트: 케미테스트 (성격 궁합 테스트)
현재 상태: 리팩토링 진행 중
- Human/Cat/Dog 3개 모드
- 총 122개 질문 (기본 43 + 심화 79)
- Big Five, Feline Five, C-BARQ 기반

기술 스택: 바닐라 JS, 단일 HTML
검증 도구: validate-questions.mjs, check-similarity.mjs
```

### 6.2 검토 요청 사항

```
1. 질문 품질 검토
   - 차원 적합성 (질문이 해당 차원을 잘 측정하는지)
   - 답변 균형 (양쪽 선택지가 동등하게 매력적인지)
   - 문화적 적절성 (한국 사용자에게 자연스러운지)

2. 확장 방향 자문
   - 추천 테스트 주제 (이상형, 업무스타일 등)
   - 과학적 기반 확보 방안
   - 바이럴 요소 강화 방법

3. 기술 구조 검토
   - 문제은행 시스템 설계
   - 유사도 검사 (Jaccard + Embedding)
   - 다중 테스트 지원 아키텍처
```

```javascript
const CHEMI_DATA = {
  human: { questions, dimensions, ... },
  cat: { ... },
  dog: { ... }
};

// 확장 구조 (제안)
const TEST_REGISTRY = {
  personality: {
    human: { ... },
    cat: { ... },
    dog: { ... }
  },
  relationship: {
    attachment: { ... },      // 애착 유형
    idealType: { ... },       // 이상형
    loveLanguage: { ... }     // 사랑의 언어
  },
  career: {
    workStyle: { ... },       // 업무 스타일
    burnout: { ... },         // 번아웃 체크
    leadership: { ... }       // 리더십 유형
  },
  fun: {
    trend2024: { ... },       // 트렌드 테스트
    meme: { ... }             // 밈 능력고사
  }
};
```

## 8. 실행 로드맵

### Phase 1: 현재 테스트 완성 (1~2주)
- [ ] 경고 수정 완료 ✅
- [ ] 질문 150개로 확장
- [ ] 유사도 검사 재실행
- [ ] 테스트 플레이 및 버그 수정

### Phase 2: 첫 번째 확장 테스트 (2~3주)
- [ ] 이상형/연애 스타일 테스트 설계
- [ ] 애착 이론 연구 및 문항 개발
- [ ] 공통 컴포넌트 추출

### Phase 3: 직장인 타겟 테스트 (2~3주)
- [ ] 업무 스타일 테스트
- [ ] 점심시간 5분 버전 최적화
- [ ] 동료 비교 기능

### Phase 4: 플랫폼화 (장기)
- [ ] 테스트 선택 허브 페이지
- [ ] 사용자 결과 히스토리
