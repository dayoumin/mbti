# 운세 콘텐츠 작성 가이드라인

> 엔터테인먼트 플랫폼 내 운세 콘텐츠의 안전한 통합을 위한 작성 원칙

---

## 1. 핵심 원칙: 엔터테인먼트 포지셔닝

### 1.1 서비스 정체성

```
우리는 "개인의 운명을 예지하는 앱"이 아니다.

우리는 "운세라는 흥미로운 소재를 매개로 사용자들이
서로 소통하고, 퀴즈를 풀며, 건전한 위로를 주고받는
엔터테인먼트 커뮤니티 플랫폼"이다.
```

### 1.2 3대 원칙

| 원칙 | 설명 | 목적 |
|------|------|------|
| **탈(脫)의료화** | 의학적 진단/치료 연상 표현 금지 | 의료법 준수 |
| **탈(脫)사행성** | 투자/도박 조장 표현 금지 | 금융법 준수 |
| **탈(脫)기만** | 100% 적중, 과학적 검증 등 금지 | 표시광고법 준수 |

## 2. 금지 표현 (Red Lines)

### 2.1 절대 금지 주제

다음 주제는 **어떠한 경우에도** 운세 콘텐츠에 포함 금지:

#### 건강/수명 관련
```
❌ "당신은 80세까지 산다"
❌ "올해 단명할 운이다"
❌ "큰 사고수가 있어 위험하다"
❌ "암이 발생할 수 있다"
❌ "당뇨를 조심해라"
❌ "이 부적을 쓰면 우울증이 낫는다"
❌ "수술수가 있습니다"
❌ "위장병을 조심하세요"
```

**이유**: 의료법 제27조(무면허 의료행위 금지) 위반 소지

#### 금융/투자 관련
```
❌ "A주식을 사세요"
❌ "로또 1등 당첨 운입니다"
❌ "부동산 투자를 지금 하세요"
❌ "코인에 투자하면 대박 난다"
❌ "이번 주 로또 번호는 00번이다"
```

**이유**: 금융소비자보호법 및 사행성 조장 우려

#### 법적/소송 관련
```
❌ "소송에서 무조건 이깁니다"
❌ "합의하는 것이 이득입니다"
❌ "법적 책임이 없습니다"
❌ "지금 합의하면 감옥에 간다"
```

**이유**: 변호사법 위반 소지

#### 관계/윤리 관련
```
❌ "당장 헤어지세요"
❌ "배우자가 바람을 피웁니다"
❌ "절대 만나선 안 될 사람입니다"
❌ 불륜 조장
❌ 범죄 모의
❌ 타인에 대한 저주
```

### 2.2 금지 키워드 필터

다음 키워드가 포함된 문장은 자동 검토 대상:

| 카테고리 | 금지 키워드 |
|----------|-------------|
| 의료 | 치료, 진단, 처방, 완치, 재활, 수술, 암, 우울증, 공황장애, 불면증 |
| 생명 | 죽음, 사망, 수명, 단명, 사고수, 흉사 |
| 금융 | 주식, 코인, 투자, 로또, 당첨, 대박, 복권 |
| 법률 | 소송, 승소, 패소, 합의, 감옥, 구속 |
| 심리 | 상담, 치유, 힐링, 클리닉, 심리치료 |

## 3. 권장 표현 (Safe Zone)

### 3.1 주제별 안전한 표현

| 주제 | ❌ 금지 표현 | ✅ 권장 표현 |
|------|-------------|-------------|
| 건강 | "위장병을 조심하세요" | "컨디션 관리가 필요한 시기입니다" |
| 건강 | "수술수가 있습니다" | "가벼운 운동으로 활력을 찾으세요" |
| 재물 | "A주식을 사세요" | "재물운의 흐름이 원활합니다" |
| 재물 | "로또 1등 당첨" | "신중한 소비가 필요합니다" |
| 재물 | "부동산 투자하세요" | "노력한 만큼 얻습니다" |
| 애정 | "당장 헤어지세요" | "새로운 인연의 기운이 있습니다" |
| 애정 | "배우자가 바람 핍니다" | "대화가 필요한 시기입니다" |
| 애정 | "절대 만나면 안됨" | "상대방을 배려하세요" |
| 법적 | "소송에서 이깁니다" | "문서상의 꼼꼼함이 요구됩니다" |
| 법적 | "합의가 이득" | "전문가의 조언을 구하는 것이 좋습니다" |

### 3.2 어미 사용 규칙

**결정론적 어미 금지:**
```
❌ "~한다" "~될 것이다" "~합니다" "~됩니다"
❌ "100% 확실합니다" "틀림없습니다"
```

**개연성 어미 권장:**
```
✅ "~할 수 있다" "~할 수도 있습니다"
✅ "~하는 경향이 있다" "~할 가능성이 보인다"
✅ "참고만 하세요" "~해보시는 건 어떨까요"
```

### 3.3 주체성(Agency) 강조

운명론적 표현보다 **사용자의 선택과 노력**을 강조:

```
❌ "당신의 운명은 정해져 있다"
✅ "당신의 노력에 따라 결과가 달라질 수 있다"

❌ "피할 수 없는 흉운이다"
✅ "이런 점을 주의하면 더 좋은 결과를 얻을 수 있다"

❌ "반드시 이 시기에 해야 한다"
✅ "이 시기가 좋은 기회가 될 수 있다"
```

### 3.4 위로와 공감 톤

해결책 제시보다 **공감과 위로** 중심:

```
✅ "요즘 힘드셨죠? 곧 좋은 일이 있을 거예요"
✅ "지금의 고민은 성장의 과정입니다"
✅ "당신의 선택을 응원합니다"
```

## 4. 면책 조항 (Disclaimer)

### 4.1 필수 삽입 위치

| 위치 | 삽입 방식 |
|------|-----------|
| 서비스 진입 시 | 팝업 또는 고정 배너 |
| 결제 단계 | 동의 체크박스 필수 |
| 결과 화면 하단 | 고정 텍스트 |

### 4.2 표준 면책 조항

```
본 서비스에서 제공하는 모든 운세 및 타로 콘텐츠는
오직 엔터테인먼트 목적으로 제공됩니다.

본 서비스의 결과는 과학적으로 검증되지 않았으며,
어떠한 경우에도 전문적인 의학적 진단, 법률적 조언,
재무적 컨설팅을 대체할 수 없습니다.

서비스 이용에 따른 모든 선택과 결정의 책임은
사용자 본인에게 있으며, 회사는 이에 대해
법적 책임을 지지 않습니다.

만약 신체적, 정신적 건강 문제가 의심될 경우
즉시 의료 전문가와 상의하십시오.
```

### 4.3 짧은 버전 (결과 화면용)

```
🎮 재미로 보는 운세입니다. 중요한 결정은 전문가와 상의하세요.
```

## 5. 콘텐츠 유형별 가이드

### 5.1 오늘의 운세

**구조:**
```
[인사/공감] + [긍정적 메시지] + [실천 가능한 조언] + [마무리 응원]
```

**예시:**
```
✅ 좋은 예시:
"오늘은 새로운 시작에 좋은 기운이 감돌아요.
평소 미뤄뒀던 일이 있다면 가볍게 시작해보세요.
작은 한 걸음이 큰 변화를 만들 수 있어요.
오늘 하루도 화이팅! 💪"

❌ 나쁜 예시:
"오늘 사고수가 있으니 조심하세요.
건강 문제가 생길 수 있습니다.
외출을 삼가고 집에 있는 것이 좋습니다."
```

### 5.2 타로 리딩

**원칙:**
- 카드 해석은 **은유적, 상징적**으로
- 구체적 예언이 아닌 **자기 성찰 유도**

**예시:**
```
✅ "이 카드는 변화의 기운을 나타내요.
    혹시 새로운 도전을 고민하고 계신가요?
    마음의 소리에 귀 기울여 보세요."

❌ "이 카드는 직장에서 해고될 것을 의미합니다.
    이번 달 안에 일어날 일이니 준비하세요."
```

### 5.3 사주/별자리 궁합

**원칙:**
- **관계 해체 조장 금지**
- 차이점보다 **조화점** 강조
- "맞지 않는다"보다 **"이런 점을 이해하면 좋다"**

**예시:**
```
✅ "두 분은 에너지 방향이 달라서
    서로에게 새로운 시각을 줄 수 있어요.
    상대방의 다른 점을 존중하면
    더 풍성한 관계가 될 수 있습니다."

❌ "두 분은 상극입니다.
    결혼하면 불행해질 것이니
    지금 당장 헤어지세요."
```

## 6. 게이미피케이션 통합

### 6.1 수집/달성 시스템

운세 결과를 **수집 가능한 디지털 자산**으로 변환:

```
예시 기능:
- 매일 다른 타로 카드 뽑아 "나만의 타로 덱" 완성
- 별자리 운세 확인 시 해당 별자리 "배지" 획득
- 연속 7일 운세 확인 시 "포춘 위크" 칭호
```

**효과**: 운세 내용(예언)보다 **행위(수집) 자체**에 집중 → 심리적 의존도 감소

### 6.2 스트릭 시스템

```
예시:
- 3일 연속 → 🌟 "럭키 스타터" 배지
- 7일 연속 → 🔮 "포춘 위크" 배지
- 30일 연속 → ✨ "운세 마스터" 배지
```

**주의**: 보상은 금전적 가치 없는 앱 내 재화로 한정 (사행성 방지)

### 6.3 운세 기반 퀴즈

일방적 전달 대신 **상호작용 유도**:

```
예시:
Q: "오늘 당신의 행운의 색깔은 무엇일까요?"
   A) 빨강  B) 파랑  C) 노랑

→ 맞추면: "정답! 오늘의 상세 운세를 확인하세요"
→ 틀리면: "아쉽지만 행운은 따로 없어요 😊 그래도 운세 볼래요?"
```

### 6.4 커뮤니티 투표

```
예시:
"이번 주 물병자리 운세가 '재물운 상승'이라는데,
 실제로 용돈 받으신 분? 🙋"

[예: 32%] [아니오: 68%]

💬 댓글:
- "월급날이라 맞는 듯 ㅋㅋ"
- "카드값만 나갔어요..."
```

**효과**: 운세 정확성을 **유머로 승화**, 커뮤니티 참여 유도

## 7. 법적 체크리스트

### 7.1 콘텐츠 작성 전

- [ ] 금지 주제(건강/수명/금융/법률) 포함되지 않음
- [ ] 금지 키워드 필터 통과
- [ ] 결정론적 어미 사용하지 않음
- [ ] 주체성(사용자 선택) 강조

### 7.2 콘텐츠 작성 후

- [ ] 의료법 위반 표현 없음 (치료, 진단, 처방 등)
- [ ] 허위/과장 광고 표현 없음 (100% 적중, 과학적 검증 등)
- [ ] 관계 해체 조장 표현 없음
- [ ] 면책 조항 적절히 배치

### 7.3 배포 전

- [ ] 결과 화면에 면책 조항 노출
- [ ] 결제 화면에 청약철회 제한 고지
- [ ] 커뮤니티 가이드라인에 유해 조언 금지 명시
- [ ] 신고 기능 작동 확인

## 9. 콘텐츠 검수 프로세스

```
작성 → [키워드 필터] → [AI 톤 검사] → [인간 검수] → 배포
         ↓                ↓              ↓
      금지어 발견?     부정적 톤?      법적 리스크?
         ↓                ↓              ↓
       자동 반려       톤 수정 요청    수정 후 재검수
```

### 9.1 자동화 검사 항목

| 검사 | 방법 |
|------|------|
| 금지 키워드 | 정규식 매칭 |
| 부정적 어미 | NLP 감정 분석 |
| 결정론적 표현 | 패턴 매칭 |

### 9.2 인간 검수 항목

| 검사 | 기준 |
|------|------|
| 맥락적 의미 | 금지 키워드가 안전한 맥락인지 |
| 톤 적절성 | 공감/위로 톤 유지 여부 |
| 법적 리스크 | 의료/금융/법률 조언 해당 여부 |
