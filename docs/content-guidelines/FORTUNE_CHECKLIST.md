# 운세 콘텐츠 체크리스트

> 콘텐츠 작성/검수 시 사용하는 빠른 참조 체크리스트

---

## Quick Check (5분 검수)

### 1. 금지 주제 확인

| 주제 | 체크 |
|------|------|
| 죽음/수명 언급 없음 | [ ] |
| 질병/의료 조언 없음 | [ ] |
| 투자/금융 조언 없음 | [ ] |
| 법률/소송 조언 없음 | [ ] |
| 관계 해체 조장 없음 | [ ] |

### 2. 금지 키워드 확인

```
Ctrl+F로 검색:
죽음, 사망, 수명, 단명, 사고수, 수술, 암, 우울증,
주식, 코인, 투자, 로또, 소송, 감옥, 치료, 진단
```

### 3. 어미 확인

| 패턴 | 체크 |
|------|------|
| "~한다/될 것이다" 없음 | [ ] |
| "100% / 확실히" 없음 | [ ] |
| "~할 수 있다/가능성" 사용 | [ ] |

### 4. 톤 확인

| 항목 | 체크 |
|------|------|
| 긍정적/공감 톤 유지 | [ ] |
| 주체성(사용자 선택) 강조 | [ ] |
| 위협/공포 조장 없음 | [ ] |

### 5. 면책 조항

| 위치 | 체크 |
|------|------|
| 결과 화면 하단에 면책 조항 있음 | [ ] |

## 상세 체크리스트

### A. 법적 준수 (필수)

#### 의료법
- [ ] "치료", "진단", "처방" 용어 없음
- [ ] 특정 질병명 언급 없음
- [ ] "심리 상담", "힐링", "치유" 표현 없음
- [ ] 건강 문제 해결 약속 없음

#### 표시광고법
- [ ] "100% 적중" 표현 없음
- [ ] "과학적으로 검증" 표현 없음
- [ ] "국내 유일", "특허" 등 최상급 없음
- [ ] 객관적 증빙 불가능한 주장 없음

#### 전자상거래법 (유료 서비스 시)
- [ ] 청약철회 제한 고지
- [ ] 자동 갱신 명시적 동의
- [ ] 해지 방법 명확히 안내

### B. 플랫폼 정책 (앱스토어 등록 시)

#### Apple App Store
- [ ] 불쾌감/공포 조장 없음 (Guideline 1.1)
- [ ] 차별화 기능 명시 (Guideline 4.3 Spam 회피)
- [ ] 앱 설명에 "엔터테인먼트" 명시

#### Google Play
- [ ] "미래 예지" 등 불가능한 약속 없음
- [ ] 수명/사망 예측 기능 없음
- [ ] 메타데이터와 실제 기능 일치

### C. 콘텐츠 품질

#### 톤 & 메시지
- [ ] 모든 결과가 긍정적 또는 건설적
- [ ] 사용자의 선택과 노력 강조
- [ ] 공감과 위로 톤 유지
- [ ] 강압적/지시적 어조 없음

#### 구조
- [ ] [인사/공감] + [메시지] + [조언] + [응원] 구조
- [ ] 면책 조항 포함
- [ ] 행운 요소(색/숫자 등) 적절히 배치

### D. 커뮤니티 안전 (소셜 기능 시)

- [ ] 유해 조언 금지 가이드라인 명시
- [ ] 신고 기능 활성화
- [ ] "전문 상담 아님" 명시

## 콘텐츠 유형별 빠른 체크

### 오늘의 운세

```
✓ 금지 주제 없음
✓ 긍정/공감 톤
✓ 개연성 어미 ("~할 수 있어요")
✓ 실천 가능한 조언
✓ 면책 조항
```

### 타로 리딩

```
✓ 은유적/상징적 해석
✓ 자기 성찰 유도
✓ 구체적 예언 없음
✓ 사용자 선택 강조
✓ 면책 조항
```

### 궁합/매칭

```
✓ 관계 해체 조장 없음
✓ 차이점 → 성장 포인트로 전환
✓ "안 맞음" 대신 "이해가 필요함"
✓ 긍정적 관계 팁 제공
✓ 면책 조항
```

## 빠른 변환 가이드

| 작성한 표현 | 안전한 대안 |
|-------------|-------------|
| "조심하세요" | "주의를 기울여 보세요" |
| "위험합니다" | "신중함이 필요해요" |
| "피하세요" | "다른 방법도 고려해보세요" |
| "반드시 하세요" | "해보시면 좋을 것 같아요" |
| "~할 것입니다" | "~할 수 있어요" |
| "확실합니다" | "가능성이 보여요" |
| "건강 주의" | "컨디션 관리가 필요해요" |
| "돈 조심" | "신중한 소비가 필요해요" |
