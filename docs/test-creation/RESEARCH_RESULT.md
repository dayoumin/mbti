# 리서치 결과 저장 포맷

> 딥리서치 결과를 이 포맷에 맞게 정리해서 `research/{subject}.md`에 저장하세요.
> AI(Claude Code)가 이 파일을 읽고 테스트 데이터를 생성합니다.

---

## 저장 위치

```
MBTI/
├── research/
│   ├── whiskey.md      # 위스키 리서치
│   ├── wine.md         # 와인 리서치
│   ├── cat.md          # 고양이 성격 리서치
│   └── ...
```

- [ ] 각 차원의 양극단(LOW/HIGH)이 명확한가?

- [ ] 모든 결과에 2-3개 조건이 있는가?

- [ ] 모든 결과에 mood, color가 있는가?

## 필드 설명

### 차원(Dimension) 필드

| 필드 | 설명 | 예시 |
|------|------|------|
| **키** | 영문 camelCase | `smoky`, `sweetness`, `openness` |
| **이름** | 한글 2-4자 | 스모키, 단맛, 개방성 |
| **이모지** | 1개 | 🔥, 🍯, 🌟 |
| **설명** | 10-20자 | "피티/스모키한 정도" |
| **LOW** | 낮은 점수 특성 | "깔끔하고 부드러운" |
| **HIGH** | 높은 점수 특성 | "강하고 스모키한" |
| **근거** | 출처/이유 | "Wishart 분류 기반" |

### 결과(Result) 필드

| 필드 | 설명 | 예시 |
|------|------|------|
| **이름** | 3-8자 | "아일라 싱글몰트" |
| **이모지** | 1개 | 🏴󠁧󠁢󠁳󠁣󠁴󠁿 |
| **한줄 설명** | 15-30자 | "강렬한 피트향의 스코틀랜드 위스키" |
| **조건** | 2-3개 차원 조건 | `smoky: high, body: high` |
| **mood** | 분위기 | happy, cool, excited, calm |
| **color** | Tailwind 색상 | `bg-slate-700` |
| **interpretation** | 심층 해석 50-100자 | - |
| **guide** | 가이드/팁 50-100자 | - |
| **matchPoints** | 추천 이유 3-4개 (매칭 테스트) | - |

### mood 옵션

| mood | 의미 | 적합한 결과 |
|------|------|------------|
| `happy` | 밝고 긍정적 | 활발한, 친근한 유형 |
| `cool` | 차분하고 세련된 | 지적인, 독립적인 유형 |
| `excited` | 열정적이고 역동적 | 도전적인, 에너지 넘치는 유형 |
| `calm` | 평화롭고 안정적 | 온화한, 편안한 유형 |
