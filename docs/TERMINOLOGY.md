## 1. 서비스 브랜딩 ✨

### 1.1 서비스명

| 구분 | 이름 | 콘셉트 |
|------|------|--------|
| **종합 서비스** | **케미 테스트** | 나와 상대의 케미를 알아보자! |
| **고양이** | **냥BTI** ❌→ **냥심 테스트** | 우리 냥이 마음 들여다보기 |

## 3. 결과 라벨 (재미있는 별명!) 🏷️

### 3.1 형식

**[특징] + [캐릭터]** 조합

### 3.2 고양이 예시

| 특성 조합 | 라벨 | 한 줄 설명 |
|-----------|------|------------|
| 호기심↑ 애교↑ | **탐험가 애교냥** | 뭐든 궁금하고 츄르면 최고! |
| 보스력↑ 경계심↑ | **츤데레 보스냥** | 이 집은 내가 지킨다냥 |
| 변덕력↑ 호기심↑ | **자유로운 영혼냥** | 오늘 기분은 나도 몰라~ |
| 애교력↑ 호기심↓ | **소파 위 귀족냥** | 움직이기 싫다냥... |
| 경계심↑ 애교↓ | **신비로운 닌자냥** | 존재감을 숨기는 중 |

### 3.3 강아지 예시

| 특성 조합 | 라벨 | 한 줄 설명 |
|-----------|------|------------|
| 에너지↑ 집사사랑↑ | **해피 바이러스댕** | 세상 모든 게 신나! |
| 집중력↑ 용감력↑ | **천재 경호댕** | 훈련도 잘하고 용감해! |
| 댕친력↑ 에너지↑ | **파티 피플댕** | 친구들아 놀자~! |
| 끈기력↑ 집중력↑ | **노력파 우등댕** | 될 때까지 한다 멍! |
| 집사사랑↑ 댕친력↓ | **집순이 애착댕** | 주인만 있으면 돼~ |

### 3.4 사람 예시

| 특성 조합 | 라벨 | 한 줄 설명 |
|-----------|------|------------|
| 인싸력↑ 모험력↑ | **열정 만수르** | 일단 GO! 생각은 나중에! |
| 계획력↑ 멘탈력↑ | **철벽 플래너** | 계획대로 착착, 흔들림 없이 |
| 공감력↑ 인싸력↑ | **분위기 메이커** | 모두를 웃게 만드는 능력자 |
| 모험력↓ 계획력↑ | **안전 제일주의** | 검증된 게 최고야 |
| 공감력↑ 인싸력↓ | **조용한 힐러** | 옆에서 묵묵히 챙겨주는 타입 |

## 5. 코드 변수명 💻

### 5.1 데이터 객체

| 기존 | **새 이름** |
|------|-------------|
| `MBTI_DATA` | `CHEMI_DATA` |
| `results_cat` | `catResults` |
| `results_dog` | `dogResults` |
| `calculateMBTI()` | `calculateChemi()` |
| `finalType` | `resultLabel` |

### 5.2 차원 코드

**사람:**
```javascript
{
  inssa: 75,      // 인싸력
  adventure: 60,  // 모험력
  empathy: 80,    // 공감력
  plan: 45,       // 계획력
  mental: 70      // 멘탈력
}
```

**고양이:**
```javascript
{
  curious: 80,    // 호기심
  alert: 40,      // 경계심
  boss: 60,       // 보스력
  random: 75,     // 변덕력
  cute: 90        // 애교력
}
```

**강아지:**
```javascript
{
  energy: 85,     // 에너지
  humanLove: 95,  // 집사 사랑
  dogFriend: 70,  // 댕친력
  focus: 60,      // 집중력
  brave: 50,      // 용감력
  persist: 65     // 끈기력
}
```
