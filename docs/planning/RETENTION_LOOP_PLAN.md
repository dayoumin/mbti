| 유형 | 트리거 | 예시 |
|------|--------|------|
| **시간 기반** | 주기적 갱신 | "오늘의 운세", "이번 주 투표", "새 퀴즈 도착" |
| **비교 기반** | 사회적 궁금증 | "다른 사람들은?", "내 또래는?", "같은 결과 사람들" |
| **변화 기반** | 자기 추적 | "6개월 전 나", "결과 변화 추이", "성장 리포트" |

## 🎯 운세 콘텐츠 역할 (퍼널 입구)

```
[운세/타로] ──→ [본 테스트] ──→ [친구 비교] ──→ [재방문 루프]
   입구           전환            핵심           습관
```

| 운세 콘텐츠 | 연결 테스트 | 연결 멘트 |
|------------|------------|----------|
| 별자리 성격 | human (Big Five) | "별자리 말고 진짜 내 성격은?" |
| 별자리 궁합 | idealType | "궁합도 좋지만, 연애 스타일은?" |
| 타로 카드 | 오늘의 추천 테스트 | "이 카드가 추천하는 테스트" |
| 띠 운세 | conflictStyle | "띠보다 정확한 대처 유형" |
