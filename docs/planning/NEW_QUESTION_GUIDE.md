# 새 질문 설계 가이드라인

> 법적으로 안전하고 과학적으로 타당한 성격 테스트 문항 설계 원칙

---

## 1. 핵심 원칙

### 1.1 절대 금지 사항

```
❌ MBTI® 관련 용어 사용
   - MBTI, Myers-Briggs
   - ENFP, ISTJ 등 4글자 코드
   - E/I, S/N, T/F, J/P 대조쌍

❌ MBTI식 대조쌍 질문
   - "파티 vs 혼자", "계획적 vs 즉흥적", "논리 vs 감정"
   - 두 가지 성향을 직접 대조하는 질문

❌ 기존 검사 문항 변형/의역
   - 번역, 의역, 유사 상황 재구성 모두 금지

❌ 추상적 자기 평가 질문
   - "당신은 외향적입니까?"
   - "창의적인 편입니까?"
```

### 1.2 반드시 지켜야 할 원칙

```
✅ 행동/관찰 기반 질문
   - 구체적이고 관찰 가능한 행동 묘사
   - "~할 때 어떻게 하나요?" 형식

✅ 단일 포인트 질문
   - 한 문항에 한 가지 행동/상황만

✅ 중립적 표현
   - 두 선택지 모두 긍정적/중립적
   - 사회적으로 바람직한 답 유도 금지

✅ 명확하고 간결한 표현
   - 짧은 문장
   - 이중부정 금지
   - 모호한 표현 제거
```

## 3. 질문 작성 템플릿

### 3.1 실제 사용 구조 (data.js)

```javascript
{
  q: "질문 텍스트?",
  dimension: "차원키",   // 아래 차원 키 목록 참조
  a: [
    { text: "높은 점수 응답", score: 5 },
    { text: "낮은 점수 응답", score: 1 }
  ]
}
```

### 3.2 차원 키(dimension) 목록

**사람 모드:**
| 키 | 이름 | 높은 점수(5) 예시 | 낮은 점수(1) 예시 |
|----|------|------------------|------------------|
| `inssa` | 인싸력 | 먼저 말 건다, 사람 많은 곳 좋아함 | 조용히 있는다, 혼자가 편함 |
| `adventure` | 모험력 | 새로운 시도, 상상력 풍부 | 검증된 방식, 현실적 |
| `empathy` | 공감력 | 감정 배려, 관계 중시 | 논리 우선, 직설적 |
| `plan` | 계획력 | 미리 준비, 체계적 | 즉흥적, 유연함 |
| `mental` | 멘탈력 | 담담함, 흔들리지 않음 | 걱정 많음, 민감함 |

**고양이 모드:**
| 키 | 이름 | 높은 점수(5) 예시 |
|----|------|------------------|
| `curious` | 호기심 | 탐색, 활동적 |
| `alert` | 경계심 | 숨거나 도망 |
| `boss` | 보스력 | 주도적, 무시 |
| `random` | 변덕력 | 불규칙, 예측불가 |
| `cute` | 애교력 | 스킨십, 반응 |

**강아지 모드:**
| 키 | 이름 | 높은 점수(5) 예시 |
|----|------|------------------|
| `energy` | 에너지 | 활발, 흥분 |
| `humanLove` | 집사 사랑 | 주인 의존, 친화적 |
| `dogFriend` | 댕친력 | 다른 개와 친함 |
| `focus` | 집중력 | 훈련 잘 따름 |
| `brave` | 용감력 | 대담, 겁 없음 |
| `persist` | 끈기력 | 포기 안 함 |

### 3.3 질문 스타일 가이드

**톤앤매너:**
- 캐주얼하고 친근한 한국어 사용 (반말/존댓말 혼용 OK)
- 이모티콘, 밈 활용 가능
- 짧고 명확한 문장

**선택지 작성:**
- 2개 선택지 (5점 vs 1점)
- 둘 다 긍정적/중립적으로 표현 (한쪽이 "나쁜 답"이 아님)
- 구체적인 행동 묘사

**실제 예시:**
```javascript
// ✅ 좋은 예시
{
  q: "친구가 '나 우울해서 빵 샀어'라고 할 때 나의 반응은?",
  dimension: "empathy",
  a: [
    { text: "무슨 빵이야? 맛있어?", score: 1 },      // 논리적
    { text: "무슨 일 있어? 괜찮아? ㅠㅠ", score: 5 } // 감정 공감
  ]
}

// ❌ 나쁜 예시 (MBTI식 대조쌍)
{
  q: "파티에서 많은 사람과 어울리기 vs 혼자 책 읽기",
  ...
}
```

## 4. 예시: 좋은 질문 vs 나쁜 질문

### 4.1 고양이 - 탐험 지수

**❌ 나쁜 예 (MBTI식)**
```
"새로운 환경에서 활발하게 탐험한다 vs 조용히 숨어있다"
→ 문제: 대조쌍 직접 비교
```

**✅ 좋은 예 (행동 기반)**
```
질문: "새로운 방에 들어갔을 때 고양이의 반응은?"
선택지:
- 구석에서 냄새만 맡다가 나온다 (1)
- 조심스럽게 한 바퀴 돌아본다 (3)
- 바로 구석구석 탐색을 시작한다 (5)
```

### 4.2 강아지 - 활력 지수

**❌ 나쁜 예 (추상적)**
```
"에너지가 넘치는 편이다"
→ 문제: 주관적 자기 평가
```

**✅ 좋은 예 (관찰 가능)**
```
질문: "산책 30분 후 강아지의 상태는?"
선택지:
- 지쳐서 쉬고 싶어한다 (1)
- 보통 컨디션이다 (3)
- 아직도 놀고 싶어한다 (5)
```

### 4.3 사람 - 소셜 에너지

**❌ 나쁜 예 (MBTI 직접 연상)**
```
"파티에서 많은 사람과 어울리기 vs 혼자 책 읽기"
→ 문제: 전형적인 MBTI 대조쌍
```

**✅ 좋은 예 (구체적 상황)**
```
질문: "주말 오후에 에너지를 충전하는 방법은?"
선택지:
- 집에서 조용히 혼자 쉰다 (1)
- 소수 친구와 카페에서 수다 (3)
- 여러 친구들과 활동적인 모임 (5)
```

## 5. 차원별 질문 예시

### 5.1 고양이 질문 예시

**경계 레이더 (Neuroticism)**
```
Q: 갑자기 큰 소리가 났을 때 고양이의 반응은?
- 숨거나 도망간다 → 5점 (높은 경계)
- 멈추고 주변을 살핀다 → 3점
- 별 반응 없다 → 1점 (낮은 경계)
```

**탐험 지수 (Extraversion)**
```
Q: 새로운 물건(택배 상자 등)이 집에 왔을 때?
- 며칠 후에야 관심을 보인다 → 1점
- 조심스럽게 접근해서 냄새를 맡는다 → 3점
- 바로 달려가서 탐색한다 → 5점
```

**주도 지수 (Dominance)**
```
Q: 다른 고양이와 먹이를 나눌 때?
- 양보하고 기다린다 → 1점
- 번갈아 먹는다 → 3점
- 먼저 먹고 지킨다 → 5점
```

**친화 지수 (Agreeableness)**
```
Q: 집사가 쓰다듬으려고 손을 뻗으면?
- 피하거나 도망간다 → 1점
- 받아들이지만 금방 떠난다 → 3점
- 몸을 비비며 더 달라고 한다 → 5점
```

### 5.2 강아지 질문 예시

**활력 지수 (Energy)**
```
Q: 하루에 산책 없이 집에 있으면?
- 조용히 쉰다 → 1점
- 가끔 놀아달라고 한다 → 3점
- 계속 뛰어다니며 안절부절한다 → 5점
```

**인간 친화도 (Human Sociability)**
```
Q: 처음 보는 사람이 집에 오면?
- 숨거나 짖으며 경계한다 → 1점
- 거리를 두고 관찰한다 → 3점
- 바로 다가가서 냄새를 맡고 인사한다 → 5점
```

**집중 지수 (Training Focus)**
```
Q: 훈련 중 간식이 보이면?
- 간식만 보고 명령을 무시한다 → 1점
- 가끔 집중이 흐트러진다 → 3점
- 명령에 집중하고 기다린다 → 5점
```

### 5.3 사람 질문 예시

**소셜 에너지 (Extraversion)**
```
Q: 일주일간 혼자만 시간을 보냈다면?
- 평화롭고 에너지가 충전된 느낌 → 1점
- 보통이다 → 3점
- 답답하고 사람들이 그립다 → 5점
```

**계획 지수 (Conscientiousness)**
```
Q: 중요한 약속 전날 준비는?
- 당일 아침에 급하게 한다 → 1점
- 전날 밤에 대충 챙긴다 → 3점
- 며칠 전부터 리스트 만들고 준비한다 → 5점
```

**공감 지수 (Agreeableness)**
```
Q: 친구의 의견과 내 의견이 다를 때?
- 내 의견을 분명히 말하고 설득한다 → 1점
- 서로 의견을 나누고 절충점을 찾는다 → 3점
- 친구 의견에 맞춰준다 → 5점
```

**법적 안전성**
- [ ] MBTI/Myers-Briggs 용어 없음
- [ ] 4글자 유형 코드 없음
- [ ] E/I, S/N, T/F, J/P 대조 없음
- [ ] MBTI식 대조쌍 아님

**질문 품질**
- [ ] 구체적 행동/상황 기반

- [ ] 두 선택지 모두 중립적
- [ ] 짧고 명확한 문장
- [ ] 이중부정 없음

**측정 타당성**
- [ ] 측정하려는 차원이 명확
- [ ] 학술 연구 특성과 연결됨
- [ ] 기존 질문과 중복 아님

**메타데이터**
- [ ] ID 부여됨
- [ ] 작성일/작성자 기록됨
- [ ] 버전 번호 있음

## 7. 결과 라벨 예시

### 7.1 점수 → 라벨 변환

**고양이**
| 차원 | 낮음 (1-2) | 중간 (3) | 높음 (4-5) |
|------|------------|----------|------------|
| 탐험 | 안락한 소파러 | 신중한 탐험가 | 호기심 대장 |
| 경계 | 태평한 고양이 | 조심성 있는 | 예민한 감시관 |
| 주도 | 순한 양 같은 | 적당히 주장하는 | 당당한 보스 |
| 친화 | 독립적인 | 때에 따라 | 애정 폭발 |

**강아지**
| 차원 | 낮음 (1-2) | 중간 (3) | 높음 (4-5) |
|------|------------|----------|------------|
| 활력 | 차분한 힐러 | 밸런스 있는 | 에너지 폭발 |
| 인간친화 | 신중한 경계견 | 선택적 친화 | 사교성 대장 |
| 집중 | 자유로운 영혼 | 학습 중인 | 집중력 천재 |

### 7.2 종합 프로필 예시

```
🐱 냥이의 성격 프로필

탐험 지수: ████████░░ 80
경계 레이더: ███░░░░░░░ 30
주도 지수: █████░░░░░ 50
친화 지수: ██████████ 100

"애정 넘치는 탐험가 냥이"

특징:
- 새로운 것에 호기심이 많아요
- 낯선 상황에도 잘 적응해요
- 집사와의 스킨십을 좋아해요

케어 팁:
- 새로운 장난감이나 환경 변화를 즐겨요
- 적극적인 교감을 원해요
```
