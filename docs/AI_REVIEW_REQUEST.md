## 1. 내 맘 테스트 (Human)

### 학술 기반
- **Big Five (OCEAN)**: Costa & McCrae (1992)
- 외향성, 개방성, 친화성, 성실성, 신경증을 친근한 이름으로 재해석

### 차원 (5개)

| 키 | 이름 | 설명 | Big Five 대응 |
|----|------|------|--------------|
| inssa | 인싸력 🎭 | 사회적 에너지와 외향성 | Extraversion |
| adventure | 모험심 🌟 | 새로움에 대한 개방성 | Openness |
| empathy | 공감력 💗 | 감정적 민감성과 배려 | Agreeableness |
| plan | 계획력 📋 | 체계성과 규칙 선호 | Conscientiousness |
| mental | 멘탈력 🧠 | 정서적 안정성 (높을수록 안정적) | Neuroticism (역) |

### 결과 라벨 (16개)

| 결과 | 조건 | 설명 |
|------|------|------|
| ⚡ 에너자이저 | inssa↑ adventure↑ empathy↑ | 어디서든 분위기를 띄우는 인싸 중의 인싸 |
| 🎯 전략가 | inssa↓ plan↑ empathy↓ | 계획적이고 논리적인 마스터마인드 |
| 💚 힐러 | empathy↑ mental↑ inssa→ | 따뜻한 마음으로 모두를 치유하는 공감왕 |
| 🚀 모험가 | adventure↑ plan↓ mental↑ | 새로운 것을 찾아 떠나는 탐험가 |
| 🛡️ 수호자 | plan↑ empathy↑ inssa↓ | 묵묵히 곁을 지키는 든든한 버팀목 |
| 🌙 몽상가 | adventure↑ empathy↑ inssa↓ | 상상의 나래를 펼치는 로맨티스트 |
| 👑 리더 | inssa↑ plan↑ mental↑ | 앞장서서 이끄는 카리스마 대장 |
| 🎨 예술가 | adventure↑ empathy↑ plan↓ | 감성과 창의력이 넘치는 자유로운 영혼 |
| 🔬 분석가 | adventure↑ empathy↓ inssa↓ | 날카로운 논리로 모든 것을 파헤치는 탐구자 |
| ☮️ 중재자 | empathy↑ inssa→ mental↑ | 갈등을 해결하고 화합을 이끄는 평화주의자 |

| 🎭 엔터테이너 | inssa↑ empathy→ plan↓ | 주목받는 것을 즐기는 무대 위의 스타 |
| 📊 현실주의자 | adventure↓ plan↑ empathy↓ | 발 딛고 있는 현실에 충실한 실용주의자 |
| 🤝 조력자 | inssa↑ empathy↑ plan→ | 누구에게나 손을 내미는 따뜻한 사람 |
| 📚 철학자 | adventure↑ inssa↓ empathy→ | 깊은 생각에 잠기는 사색가 |
| ⚖️ 균형잡힌 사람 | 모두 medium | 모든 면에서 조화를 이루는 밸런스 마스터 |

### 질문 예시

```

- 친한 친구 옆에 딱 붙어서 조용히 있는다 (inssa +1)

Q: 멍 때릴 때 주로 하는 생각은?
- 오늘 저녁 뭐 먹지? 내일 할 일이 뭐더라? (adventure +1)
- 좀비 사태가 터지면 어디로 도망가지? 로또 당첨되면... (adventure +5)
```

## 2. 냥심 테스트 (Cat)

### 학술 기반
- **Feline Five**: Litchfield et al. (2017)
- 고양이 성격의 5가지 차원을 학술적으로 검증한 연구

### 차원 (5개)

| 키 | 이름 | 설명 | Feline Five 대응 |
|----|------|------|-----------------|
| curious | 호기심 🔍 | 새로운 것에 대한 탐구심 | Curiosity |
| alert | 경계심 👀 | 환경에 대한 민감도 (높을수록 경계심 강함) | Fearfulness (역) |
| boss | 보스기질 👑 | 독립성과 지배력 | Dominance |
| random | 엉뚱함 🎲 | 예측불가한 행동 패턴 | Impulsiveness |
| cute | 애교력 🥰 | 친밀감과 사회성 | Friendliness |

### 결과 라벨 (10개)

| 결과 | 조건 | 설명 |
|------|------|------|
| 🌙 철학 냥이 | curious↑ alert→ boss↓ | 창밖을 보며 우주의 섭리를 생각해요 |
| 👑 보스 냥이 | boss↑ alert→ cute↓ | 이 집의 주인은 나다옹! |
| 🎉 인싸 냥이 | cute↑ curious↑ alert↓ | 새로운 친구를 만나면 먼저 다가가요 |
| 😇 천사 냥이 | cute↑ boss↓ alert↓ | 집사가 힘들면 옆에서 골골송을 불러줘요 |
| 🔭 탐험 냥이 | curious↑ random↑ alert↓ | 새로운 곳은 모두 나의 탐험지! |
| 🚨 경계 냥이 | alert↑ boss→ cute↓ | 낯선 것에는 일단 하악! |
| 🎲 엉뚱 냥이 | random↑ curious↑ boss→ | 다음 행동은 나도 모른다옹! |
| 💅 도도 냥이 | boss↑ cute↓ random↓ | 쓰다듬는 것은 허락할 때만 가능 |
| 🥰 애교 냥이 | cute↑ random→ boss↓ | 집사 무릎이 내 자리다옹~ |
| ☯️ 균형 냥이 | 모두 medium | 상황에 따라 적절히 행동하는 중용의 고양이 |

### 질문 예시

```
Q: 새로운 장난감을 발견했다!
- 바로 달려가서 냄새를 맡고 탐색한다 (curious +5)
- 멀리서 지켜보다가 나중에 확인한다 (curious +1)

Q: 집사가 집에 오면?
- 문 앞에서 기다렸다가 마중 나간다 (cute +5)
- 자기 자리에서 눈만 살짝 뜬다 (cute +1)
```

## 3. 댕심 테스트 (Dog)

### 학술 기반
- **C-BARQ**: Hsu & Serpell (2003), 업데이트 (2022)
- Canine Behavioral Assessment and Research Questionnaire

### 차원 (6개)

| 키 | 이름 | 설명 | C-BARQ 대응 |
|----|------|------|------------|
| energy | 활력 ⚡ | 에너지 수준과 활동성 | Excitability |
| humanLove | 인간사랑 ❤️ | 사람에 대한 친화력 | Attachment |
| dogFriend | 동료애 🐕 | 다른 강아지와의 사회성 | Dog Sociability |
| focus | 집중력 🎯 | 훈련 가능성과 주의력 | Trainability |
| brave | 용감함 🦁 | 두려움에 대한 반응 (높을수록 대담함) | Fearfulness (역) |
| persist | 끈기 💪 | 목표를 향한 집념 | Persistence |

### 결과 라벨 (10개)

| 결과 | 조건 | 설명 |
|------|------|------|
| ⚡ 에너자이저 멍멍이 | energy↑ humanLove↑ dogFriend↑ | 지치지 않는 무한 에너지의 소유자 |
| ❤️ 집사바라기 멍멍이 | humanLove↑ brave→ dogFriend↓ | 보호자가 세상의 전부인 충견 |
| 👑 리더 멍멍이 | brave↑ persist↑ energy↑ | 무리를 이끄는 타고난 리더 |
| 🧠 천재 멍멍이 | focus↑ persist↑ energy→ | 한 번 보면 다 기억하는 똑똑이 |
| 🐾 겁쟁이 멍멍이 | brave↓ humanLove↑ energy↓ | 세상이 무서워... 보호자만 믿어요 |
| 🎉 파티 멍멍이 | energy↑ dogFriend↑ humanLove↑ | 어디서든 분위기 메이커 |
| 🐂 고집쟁이 멍멍이 | persist↑ brave↑ focus↓ | 내 방식대로 할 거야! |
| 😴 느긋이 멍멍이 | energy↓ persist↓ brave→ | 급할 거 뭐 있어? 천천히~ |
| 🛡️ 수호자 멍멍이 | brave↑ humanLove↑ focus↑ | 가족을 지키는 든든한 보디가드 |
| ⚖️ 균형 멍멍이 | 모두 medium | 모든 면에서 조화로운 이상적인 반려견 |

### 질문 예시

```
Q: 산책 중 다른 강아지를 만나면?
- 꼬리 흔들며 다가가서 인사한다 (dogFriend +5)
- 보호자 뒤에 숨거나 무시한다 (dogFriend +1)

Q: 새로운 훈련을 시작하면?
- 집중해서 빠르게 배운다 (focus +5)
- 금방 지루해하며 딴 짓을 한다 (focus +1)
```

## 4. 토끼 맘 테스트 (Rabbit)

### 학술 기반
- Rabbit Behavior Assessment 연구
- 토끼 행동학 논문들 기반

### 차원 (5개)

| 키 | 이름 | 설명 |
|----|------|------|
| curious | 호기심 🔍 | 새로운 것에 대한 탐구심 |
| social | 사교성 💕 | 사람/동물과의 친밀도 |
| active | 활동성 🏃 | 움직임과 놀이 선호도 |
| brave | 담력 🦁 | 낯선 상황에 대한 대담함 |
| chill | 느긋함 😌 | 스트레스 관리 능력 |

### 결과 라벨 (10개)

| 결과 | 조건 | 설명 |
|------|------|------|
| 🗺️ 탐험가 토끼 | curious↑ social↑ brave↑ | 호기심 만땅! 뭐든 궁금한 탐험가 |
| 🥰 애교쟁이 토끼 | social↑ chill↑ | 사람 좋아! 스킨십 최고! |
| 🏃 액티브 토끼 | active↑ brave↑ | 뛰고 뛰고 또 뛰고! 빙키 마스터 |
| 🤔 신중한 토끼 | curious↓ brave↓ | 일단 지켜본다... 안전 제일! |
| 😎 독립적인 토끼 | social↓ chill↑ | 나는 나! 쿨한 마이웨이 |
| 🐰 겁쟁이 토끼 | brave↓ chill↓ | 무서워... 조용히 해줘... |
| 📚 사색가 토끼 | active↓ chill↑ | 조용히 생각 중... 철학 토끼 |
| ⚡ 에너자이저 토끼 | active↑ chill↓ | 쉴 틈이 없어! 에너지 폭발! |
| 🌟 사교스타 토끼 | social↑ active↑ curious↑ | 모두와 친해지는 인기쟁이! |
| ⚖️ 밸런스 토끼 | 모두 medium | 모든 면에서 균형 잡힌 이상적인 토끼 |

### 질문 예시

```
Q: 새로운 장난감을 케이지에 넣어주면?
- 바로 다가가서 코로 킁킁, 이빨로 콕콕 탐색한다 (curious +5)
- 구석에서 경계하다가 한참 후에 확인한다 (curious +1)

Q: 케이지 밖으로 나오면 주로?
- 신나게 뛰어다니고 빙키(공중 점프+몸 비틀기)한다 (active +5)
- 한 곳에 납작 엎드려서 쉰다 (active +1)

Q: 손을 내밀면?
- 코를 가져다 대고 냄새 맡고 핥아준다 (social +5)
- 뒷발로 쿵쿵 경고하며 피한다 (social +1)
```

## 6. 연애 이상형 테스트 (IdealType) - 신규

### 학술 기반
- **Sternberg 삼각 이론 (1986)**: 열정(Passion), 헌신(Commitment)
- **애착 이론 (Hazan & Shaver, 1987)**: 친밀감 편안함
- **5 Love Languages (Chapman, 1992)**: 표현, 함께하는 시간

### 테스트 유형
- **매칭형 테스트**: 성격 분석이 아닌 "이상형 추천" 형식
- **resultFormat**: `matching` (matchPoints 표시)

### 차원 (5개)

| 키 | 이름 | 설명 | 학술 근거 |
|----|------|------|----------|
| passion | 열정 🔥 | 연애 온도, 설렘 정도 | Sternberg - Passion |
| commit | 헌신 💍 | 관계 진지도, 미래 지향 | Sternberg - Commitment |
| close | 친밀 🤗 | 밀착 정도, 거리감 | 애착이론 |
| express | 표현 💬 | 감정 표현 방식 | 5 Love Languages |
| active | 활동 🎯 | 데이트 스타일 | 5 Love Languages |

### 결과 라벨 (8개)

| 결과 | 조건 | 설명 |
|------|------|------|
| 🥰 다정다감 연인 | passion↑ express↑ close↑ | 따뜻한 말과 애정 표현이 가득한 상대 |
| 🏔️ 든든한 버팀목 | commit↑ close↑ | 믿음직하고 헌신적인 상대 |
| 💘 열정적인 로맨티스트 | passion↑ active↑ | 설렘과 열정이 넘치는 상대 |
| 🦋 자유로운 동반자 | close↓ commit→ | 서로의 공간을 존중하는 상대 |
| ☕ 편안한 베스트프렌드 | passion↓ close→ express↓ | 친구처럼 편안한 상대 |
| ⚡ 액티브 파트너 | active↑ passion→ | 함께 활동하는 것을 즐기는 상대 |
| ✨ 진지한 소울메이트 | commit↑ express↑ | 깊은 대화와 교감을 나누는 상대 |
| ⚖️ 밸런스 연인 | 기본 | 모든 면에서 균형 잡힌 상대 |

### 질문 예시

```
Q: 연인에게 받고 싶은 연락 빈도는?
- 수시로! 뭐해? 밥 먹었어? (passion +5)
- 필요할 때만 하면 돼 (passion +1)

Q: 연애를 시작할 때 나는?
- 미래까지 생각하고 시작해 (commit +5)
- 일단 만나보고 결정해 (commit +1)

Q: 이상적인 데이트는?
- 맛집, 카페, 영화... 밖으로! (active +5)
- 집에서 넷플릭스가 최고 (active +1)
```

### 결과 화면 특징 (matchPoints)

```
💘 이런 사람이 맞아요
✓ 매일 연락하며 안부를 묻는 사람
✓ 스킨십과 애정 표현이 풍부한 사람
✓ 함께하는 시간을 소중히 여기는 사람
```

## 7. 반려식물 매칭 테스트 (Plant) - 신규

### 학술 기반
- **NASA Clean Air Study (1989)**: 공기정화 식물 연구
- **원예치료 연구 (Söderback et al., 2004)**: 식물과 웰빙
- **RHS (Royal Horticultural Society)**: 식물 관리 난이도 분류

### 테스트 유형
- **매칭형 테스트**: 사용자 환경에 맞는 식물 추천
- **resultFormat**: `matching` (matchPoints 표시)

### 차원 (5개)

| 키 | 이름 | 설명 | 참고 기준 |
|----|------|------|----------|
| care | 관리력 🌱 | 식물 돌봄 시간/관심 | 관리 난이도 |
| light | 채광 ☀️ | 집의 햇빛 환경 | RHS 광량 요구 |
| water | 물주기 💧 | 물주기 습관 | 관수 빈도 |
| space | 공간 🏠 | 식물 배치 공간 | 식물 크기 |
| style | 취향 🎨 | 선호 스타일 | 개화/관엽 |

### 결과 라벨 (10개)

| 결과 | 조건 | 설명 |
|------|------|------|
| 🌿 몬스테라 | care↑ light→ space↑ | 존재감 있는 이국적 식물 |
| 🌵 스투키 | care↓ water↓ | 물 안 줘도 OK! 초보자 친구 |
| 💚 포토스 | light↓ care→ | 어디서든 잘 자라는 만능 식물 |
| 🍃 스킨답서스 | space↓ light↓ care↓ | 행잉 플랜트의 정석 |
| 🪴 다육이 | water↓ space↓ style↓ | 통통 귀여운 미니 식물 |
| 🌴 아레카야자 | care↑ light↑ space↑ | 열대 휴양지 느낌 |
| 🗡️ 산세베리아 | care↓ light→ | 밤에도 산소 뿜는 침실 식물 |
| 🌿 허브 | care↑ light↑ style↑ | 키우고 먹고 향기까지! |
| 🌳 홍콩야자 | care→ space↑ light→ | 우아한 공기정화 식물 |
| 🎨 칼라테아 | care↑ water↑ style↑ | 예술작품 같은 무늬잎 |

### 질문 예시

```
Q: 평소 식물 관리에 얼마나 시간을 쓸 수 있나요?
- 매일 돌봐줄 수 있어요 (care +5)
- 일주일에 한번 정도만... (care +1)

Q: 집에 햇빛이 얼마나 들어오나요?
- 온종일 환하게 들어와요 (light +5)
- 어둡고 그늘진 편이에요 (light +1)

Q: 원하는 식물 크기는?
- 존재감 있는 큰 식물! (space +5)
- 아담한 미니 식물 (space +1)
```

### 결과 화면 특징 (matchPoints)

```
🌱 이런 분에게 추천해요
✓ 초보 식집사
✓ 자주 집을 비우는 분
✓ 물주기 자주 잊는 분
✓ 사무실에 두고 싶은 분
```

## 점수 시스템

### 점수 계산
- 각 선택지: **5점** (해당 차원 높음) 또는 **1점** (해당 차원 낮음)
- 차원별 백분율: `(획득 점수 / 최대 점수) × 100`

### 레벨 판정
- **high (↑)**: 60% 이상
- **medium (→)**: 40% ~ 60%
- **low (↓)**: 40% 이하

### 결과 매칭
1. **완전 일치**: 모든 조건이 일치하는 결과 반환
2. **부분 매칭**: 완전 일치 없을 경우
   - 각 결과별로 조건 일치 개수 계산
   - 가장 많이 일치하는 결과 선택
   - 동점 시 먼저 정의된 결과 우선
3. **폴백**: 어떤 조건도 일치하지 않으면 "균형" 결과 반환

> **참고**: 극단적 조합(모든 차원 high/low)은 대부분 부분 매칭으로 처리되며, 각 결과의 핵심 차원(첫 번째 조건)을 기준으로 가장 적합한 결과가 선택됩니다.
