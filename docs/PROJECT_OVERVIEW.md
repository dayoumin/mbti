# 케미 테스트 앱 - 프로젝트 개요

> ⚠️ **법적 준수**: 본 프로젝트는 MBTI®(등록상표)를 사용하지 않습니다.
> **Big Five (OCEAN)** 및 **IPIP 공개 도메인** 프레임워크를 기반으로 합니다.

## 프로젝트 비전

> **"재미있고 유용한 성격 탐색 도구"**
>
> 과학적 연구를 참고하되, "재미로 보는 테스트"로 포지셔닝하여
> 자기 이해와 관계 개선에 도움을 주는 서비스

## 2. 핵심 목표

### 2.1 주요 목표

1. **다중 대상 성격 테스트**
   - 사람: 내 맘 테스트 🧑
   - 고양이: 냥심 테스트 🐱
   - 강아지: 댕심 테스트 🐕

2. **프로필 저장 및 관리**
   - 여러 대상의 테스트 결과 저장
   - "오늘은 우리 펫, 내일은 아이" 시나리오 지원

3. **관계별 케미 분석**
   - 사람-사람: 연애, 부부, 직장, 가족
   - 사람-펫: 주인과 반려동물 케미

4. **맞춤형 조언 제공**
   - 관계 유형별 세분화된 조언
   - 펫 케어 가이드

### 2.2 차별화 포인트

| 기존 서비스 | 우리 앱 |
|------------|---------|
| 단일 테스트 후 끝 | 프로필 저장 → 케미 분석 |
| 사람만 대상 | 사람 + 고양이 + 강아지 |
| 일반적인 조언 | 관계 유형별 맞춤 조언 |
| 결과만 제공 | 스펙트럼 비율 표시 |

## 3. 대상 사용자

### 3.1 Primary User

| 세그먼트 | 특징 | 니즈 |
|----------|------|------|
| MZ세대 (20-35) | 성격 테스트에 높은 관심 | 재미, 공유, 자기 표현 |
| 반려동물 보유자 | 펫 성격 궁금 | 케어 방법, 케미 |
| 커플/부부 | 관계 개선 관심 | 소통 방법, 이해 |

### 3.2 Use Cases

```
1. "내 성격 테스트 → 저장 → 친구들과 비교"
2. "우리 고양이 냥심 테스트 → 나와의 케미 확인 → 케어팁 적용"
3. "남자친구 테스트 → 커플 케미 → 연애 조언 참고"
4. "우리 아이 테스트 → 부모-자녀 케미 → 양육 스타일 조정"
5. "상사 성격 추정 → 직장 케미 → 소통 방법 참고"
```

## 5. 데이터 신뢰성 전략

### 5.1 포지셔닝

```
❌ "과학적으로 검증된 심리 진단 도구"
✅ "재미로 보는 성격 테스트 🔮 (학술 연구 참고)"
```

### 5.2 신뢰성 확보 방법

| 방법 | 설명 |
|------|------|
| **투명성** | "재미를 위한 테스트입니다" 명시 |
| **스펙트럼 표시** | 이분법 ❌ → 인싸력 80점 / 계획력 45점 비율 |
| **연구 참고** | Big Five, C-BARQ, Feline Five 기반 |
| **출처 제공** | 관심 있는 사용자를 위한 학술 링크 |

### 5.3 참고 연구

| 대상 | 연구 | 특성 수 |
|------|------|---------|
| 사람 | Big Five (OCEAN) | 5 |
| 고양이 | Feline Five (2017) | 5 |
| 고양이 | 헬싱키대 (2021) | 7 |
| 강아지 | C-BARQ (2022) | 7 |

→ 자세한 내용: [research/REFERENCES.md](./research/REFERENCES.md)

## 6. 관계 유형 세분화

### 6.1 사람 관계

| 대분류 | 세분류 | 조언 차별화 |
|--------|--------|-------------|
| **연애** | 썸, 연인, 장거리 | 소통 방식, 데이트 스타일 |
| **부부** | 신혼, 육아기, 황혼 | 갈등 해결, 역할 분담 |
| **가족** | 부모-자녀, 형제 | 양육법, 이해 방법 |
| **직장** | 상사-부하, 동료 | 업무 스타일, 피드백 |
| **친구** | 친한 친구, 그룹 | 놀이 스타일, 갈등 관리 |

### 6.2 펫 관계

| 관계 | 조언 내용 |
|------|-----------|
| 주인-고양이 | 케미 분석, 돌봄 팁, 스트레스 관리 |
| 주인-강아지 | 훈련 방법, 산책 스타일, 놀이 방식 |
