## Tags 작성 규칙

### 패턴
1. **첫 번째 태그**: 카테고리명 (고양이, 강아지, 토끼, 식물)
2. **2-4번째 태그**: 핵심 키워드 (주제, 행동, 상태 등)
3. **마지막 태그**: 분류 또는 용도 (건강, 관리, 감정표현 등)

### 예시
```typescript
// cat-k-001: 꼬리 행동 퀴즈
tags: ['고양이', '꼬리', '행동', '감정표현']

// dog-k-001: 위험 음식 퀴즈
tags: ['강아지', '음식', '위험', '포도', '건강']

// plant-k-002: 몬스테라 퀴즈
tags: ['식물', '몬스테라', '잎', '진화', '빛']
```
