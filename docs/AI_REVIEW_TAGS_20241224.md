## 2. 태그 작성 규칙

1. **첫 번째 태그는 카테고리 명칭**: 고양이/강아지/토끼/식물 등.
2. **두 번째~네 번째 태그는 핵심 주제**: 행동, 건강, 습성, 식단 등.
3. **마지막 태그는 분류 보조**: 의료, 예방접종, 위험, 관리 등.
4. **중복 태그는 제거**: 같은 의미의 태그가 겹치지 않도록 정리.

예시:
```ts
// cat-k-001
tags: ['고양이', '꼬리', '행동', '감정표현']

// dog-k-001
tags: ['강아지', '음식', '위험', '포도', '건강']

// plant-k-002
tags: ['식물', '몬스테라', '진화', '빛']
```

## 5. 태그 일관성 분석

- **의미 중복 가능성**: `백신` vs `예방접종`, `건강` vs `의료`, `감정표현` vs `애정표현`
- **명확도 차이**: `위험`과 `독성`의 구분 기준이 모호할 수 있음
- **액션 동사 혼용**: `관리`/`양육`/`체크` 등 유사 범주의 태그가 혼재
- **개선 제안**: 동의어 그룹을 묶어 대표 태그로 정규화하고, 세부 태그는 보조로 제한
